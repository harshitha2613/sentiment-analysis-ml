# sentiment-analysis-ml
Election Voter Management using Sentiment Analysis

This project focuses on analyzing public opinion during elections using sentiment analysis. It includes visualization, word cloud generation, and interactive prediction.

Project Description

The system analyzes text data collected from social media (Twitter-like dataset) and classifies it into positive, negative and neutral sentiments. It also provides visual insights using graphs and maps.

Features

- Sentiment classification using NLP  
- State-wise sentiment visualization (India)  
- District-level analysis (basic simulation)  
- Word cloud generation  
- Graphs and heatmaps  
- Real-time user input for sentiment prediction  

Tools and Technologies

Python, Pandas, NumPy, TextBlob, Scikit-learn, Matplotlib, Seaborn, Plotly  

Note

Due to API limitations, a sample dataset is used to simulate Twitter data.

Result

The model successfully classifies sentiment and provides visual insights using charts and maps.
